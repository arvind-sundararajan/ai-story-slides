Introducing LightLLM

LightLLM is a Python-based Large Language Model inference and serving framework that leverages the strengths of numerous well-regarded open-source implementations, including FasterTransformer, TGI, vLLM, and FlashAttention. The framework has a pure-python design and token-level KC Cache management, making it easy to use as the basis for research projects. The latest features of LightLLM include Prefix KV Cache Transfer between DP rankers, which is now supported.
