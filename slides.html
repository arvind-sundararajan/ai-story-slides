Introduction

OpenAI built the OpenAI Agents SDK.

The OpenAI Agents SDK is a powerful framework for building multi-agent workflows, supporting the OpenAI Responses and Chat Completions APIs, as well as over 100 other large language models. The core concepts of the OpenAI Agents SDK include agents, which are large language models configured with instructions, tools, guardrails, and handoffs, allowing them to delegate tasks to other agents. Agents can also be used as tools, enabling them to take actions through various functions, including hosted tools. The OpenAI Agents SDK features configurable safety checks, known as guardrails, for input and output validation, as well as built-in mechanisms for involving humans across agent runs, referred to as human in the loop.

The OpenAI Agents SDK has a simple architecture, with agents being the core component, and it provides a range of key APIs, including the Agent and Runner classes, which can be used to create and run agents. The performance of the OpenAI Agents SDK is dependent on the underlying large language models and the specific use case, but it is designed to be scalable and efficient. The OpenAI Agents SDK can be used in a variety of production use cases, including building conversational interfaces, automating workflows, and developing voice assistants. The SDK is designed to be provider-agnostic, allowing users to work with a range of large language models.

The OpenAI Agents SDK is built on top of several open-source libraries, including Pydantic, PydanticAI, LiteLLM, MkDocs, Griffe, uv, and ruff, and it is committed to continuing to build the Agents SDK as an open-source framework. To get started with the OpenAI Agents SDK, users must set up a Python environment with Python 3. 10 or newer and install the OpenAI Agents SDK package using pip. The package can be installed with optional groups, including voice and redis, to support voice and Redis session functionality.

The OpenAI Agents SDK can be used to run agents synchronously or asynchronously, and users can explore the examples directory and documentation to learn more about the SDK's features and capabilities.

This book is intended for developers and researchers who want to learn how to use the OpenAI Agents SDK to build powerful multi-agent workflows. Through this book, readers will learn how to create and run agents, use the various APIs and tools provided by the SDK, and deploy their workflows in production environments. The book will cover the key concepts and features of the OpenAI Agents SDK, including agents, tools, guardrails, and human in the loop, as well as the various use cases and applications of the SDK. By the end of this book, readers will have a deep understanding of the OpenAI Agents SDK and be able to use it to build complex and powerful multi-agent workflows.
