Introduction

Flyte was built by the Flyte team at flyteorg.

Flyte is an open-source orchestrator that facilitates building production-grade data and ML pipelines, leveraging Kubernetes as its underlying platform. It is built for scalability and reproducibility, allowing user teams to construct pipelines using the Python SDK and seamlessly deploy them on both cloud and on-premises environments. Flyte enables distributed processing and efficient resource utilization. The platform offers a range of features for building and managing data and machine learning workflows, including strongly typed interfaces to validate data at every step of the workflow, and supports development in any language, with SDKs available for Python, Java, Scala, and JavaScript.

Flyte was created to solve the problem of building and managing complex data and ML pipelines, providing a robust and scalable solution for teams to construct and deploy pipelines. The latest features of Flyte include a robust type engine, allowing users to write code in Python or any other language. The Flyte architecture is based on Kubernetes, enabling scalability and reproducibility. Key APIs include the Python SDK, which allows users to construct and deploy pipelines.

In terms of performance, Flyte is designed for distributed processing and efficient resource utilization, making it suitable for large-scale data and ML pipelines.

The Flyte project is hosted on GitHub, with a repository located at flyteorg/flyte, and a separate repository for the Flyte SDK located at flyteorg/flyte-sdk. The project is licensed under the Apache 2. 0 license and has passed the OpenSSF best practices. Flyte also has a Helm chart available on Artifact Hub.

For production use cases, Flyte provides a range of features and tools, including support for distributed processing, efficient resource utilization, and scalability. Flyte can be used in a variety of environments, including cloud and on-premises, and is designed to integrate with existing workflows and pipelines.

This book is targeted towards data engineers, machine learning engineers, and DevOps teams who want to learn how to build and manage data and ML pipelines using Flyte. Readers will learn how to construct pipelines using the Python SDK, deploy them on cloud and on-premises environments, and leverage Flyte's features for scalability, reproducibility, and efficient resource utilization. By the end of this book, readers will have a deep understanding of Flyte's architecture, features, and use cases, and will be able to apply this knowledge to build and manage complex data and ML pipelines in production environments.
