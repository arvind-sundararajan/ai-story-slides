What yao Is and Why It Matters

yao is an autonomous agent engine

Yao is an open-source engine for building autonomous agents, characterized by its event-driven, proactive, and self-scheduling architecture. Yao's autonomous agent framework allows for the creation of agents that work like real team members, with three trigger modes, including clock, human, and event-based triggers. Yao features a unique entry point system, where agents can be triggered by email, events, and scheduled tasks, rather than traditional chatbox interactions. Yao's architecture enables efficient and scalable execution of autonomous agents, with a focus on proactive and self-scheduling capabilities.

Yao supports multi-agent orchestration, allowing agents to delegate, collaborate, and compose dynamically.

Yao has native support for Mission Control Protocol (MCP) tools, enabling seamless integration without the need for adapters. This is achieved through process transport, which maps Yao processes directly to MCP tools, as well as external server connections via Server-Sent Events (SSE) or Standard Input/Output (STDIO). Yao's platform also features schema mapping, allowing for declarative input/output schemas. The engine includes a built-in GraphRAG, which provides vector search capabilities using OpenAI and FastEmbed, as well as knowledge graph and entity-relationship retrieval.

Yao's full-stack runtime allows for everything to be included in a single executable, featuring an all-in-one data, API, agent, and UI engine. The platform supports TypeScript and includes a built-in V8 engine, eliminating the need for Node. js, Python, or containers. This results in a single binary that can run on edge devices, including ARM64 and x64 devices.

Yao has various production use cases, including the development of admin dashboards, such as Yao Admin, which provides a code-free programming experience using JSON to create database models and describe interfaces.
